# Koona Manoj Kumar — Portfolio

A production-ready personal portfolio website built with Next.js 15, TypeScript, Tailwind CSS, and Framer Motion.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: Vercel

## Design

- Dark glassmorphism aesthetic
- Smooth scroll animations with Framer Motion
- Radial gradient backgrounds with ambient orbs
- Animated hero with role tags
- Interactive project case studies (expandable)
- Skill progress bars
- Timeline-based experience section
- Responsive across all breakpoints

## Sections

1. Hero — Animated name, role tags, CTAs, social links
2. About — Bio, stats, highlight cards
3. Skills — Categorized skill bars + tech tags
4. Experience — Timeline with AIESEC, CR, Event Coordinator
5. Projects — SHE SHIELD & PURE CITY full case studies
6. Leadership — AIESEC, CR, Event Coordinator, Content Creator
7. Certifications — IBM, Cisco, Infosys credentials
8. Achievements — SOG 2023, Paper Presentation Runner-Up, Hackathons
9. Contact — Form + social links
10. Footer

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy to Vercel

```bash
npx vercel
```

Or connect your GitHub repo to [vercel.com](https://vercel.com).

## Customization

All data lives in `data/index.ts` — update projects, skills, experience, and personal info there.
